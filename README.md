To Do List App

A simple To-Do List web application to add, manage, and delete tasks efficiently.

 Features:
- Add new tasks
- Delete tasks
- Mark tasks as completed (if implemented)
- Simple and user-friendly interface

 Tech Used:
- HTML
- CSS
- JavaScript
